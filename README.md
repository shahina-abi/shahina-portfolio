# Shahina Habeeb - Portfolio Website

A modern, interactive portfolio website built with **React + Vite**, **Tailwind CSS**, and **GSAP** animations.

## 🚀 Tech Stack

- **Framework:** React + Vite
- **Styling:** Tailwind CSS (v4)
- **Animations:** GSAP (GreenSock Animation Platform)
- **Smooth Scrolling:** Lenis
- **Icons:** Lucide React

## 🛠️ Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    cd shahina-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run Development Server:**
    ```bash
    npm run dev
    ```

4.  **Build for Production:**
    ```bash
    npm run build
    ```

## ✨ Features

- **Hero Section:** Animated text reveal and parallax effects.
- **Smooth Scrolling:** Integrated Lenis for a premium feel.
- **Interactive UI:** Hover effects, 3D tilts, and scroll-triggered animations.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop.
- **Contact Form:** Ready-to-use UI for contact inquiries.

## 📂 Project Structure

- `src/components`: Reusable components (Hero, About, Skills, Projects, etc.)
- `src/index.css`: Global styles and Tailwind configuration.
- `src/App.jsx`: Main application entry point.
